# ROS 2 Mobile Manipulator

A ROS 2 + Gazebo Harmonic simulation of a differential‑drive mobile robot equipped with a 2‑DOF manipulator arm.  
This project includes URDF/Xacro modeling, Gazebo simulation, ROS 2 control, TF, RViz visualization, and ROS ↔ Gazebo topic bridging.

---

## Features
- Differential‑drive mobile base
- 2‑DOF manipulator arm
- URDF/Xacro modular robot description
- Gazebo Harmonic simulation
- ROS 2 Control integration
- JointState + TF publishing
- RViz visualization
- ROS ↔ Gazebo bridge for motion control
- Teleoperation for base and arm
